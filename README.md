# Berlekamp Algorithm for Polynomial Factorisation over Finite Fields

**Bachelor's Thesis — BSc Mathematics | Universidad de Valladolid (2026)**  
**Grade: 9/10**

---

### Overview

Theoretical and computational study of the Berlekamp algorithm for 
factoring polynomials over finite fields. This project bridges abstract 
algebra with computational mathematics, covering both the rigorous 
mathematical foundations and a complete Python implementation.

The Berlekamp algorithm has direct applications in cryptography and 
error-correcting codes — fields where finite field arithmetic is essential.

---

### Project highlights

- Full development of the algebraic foundations: finite fields, 
  irreducible polynomials, field extensions.
- Step-by-step construction of the Berlekamp matrix via the 
  Frobenius operator.
- Explicit factorisation examples over F2, F4 and F5.
- Computational complexity analysis.
- Complete Python implementation in Jupyter Notebook.

---

### Applications

- Cryptography (RSA, elliptic curve cryptography)
- Error-correcting codes (Reed-Solomon, BCH codes)
- Computer algebra systems
